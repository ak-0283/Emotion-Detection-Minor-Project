Emotion Detection App (Windows PowerShell Setup + Run)

1) Open PowerShell in this project folder:
	C:\

2) Create and activate a virtual environment:
	python -m venv .venv
	.\.venv\Scripts\Activate.ps1

3) Upgrade pip:
	python -m pip install --upgrade pip

4) Install required packages:
	pip install streamlit tensorflow numpy opencv-python pillow keras

5) Run the app (IMPORTANT: use Streamlit, not python app.py):
	streamlit run app.py

6) Open in browser:
	http://localhost:8501

If "streamlit" command is not recognized, use:
	python -m streamlit run app.py

If PowerShell blocks activation script, run once:
	Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

Notes:
- `python app.py` is not the correct launch command for this project.
- This is a Streamlit web app, so it must be started with `streamlit run app.py`.
